dangle.js Examples

Tested on the latest version of Chrome, Safari, and Firefox. The examples will also adapt to mobile devices (tested on Nexus 7 (tablet) and HTC One X (phone), both running Chrome. This runs completely in the browser so just view source to see how it works. Graphs are interactive and animated. Clicking on any graph will modify the data, causing everything to update.

dangle is a set of AngularJS directives that provide common D3 visualizations for elasticsearch.
